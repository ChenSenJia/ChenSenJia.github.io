    • 实现细节
      • 正则表达式为[a-zA-Z]\w*，匹配到以字母开头的变量和关键字，但是不会匹配到运算符，这样的好处是使得后续的keywords不繁杂，免去了对运算符的考虑。
      • 正则匹配使用re.sub()，其中第二个变量以函数的形式给出！该函数接受每一次正则表达式在第三个参数那里匹配到的中间结果，以其作为参数，处理之后返回结果为字符串，替换匹配到的位置上。这个函数使得处理匹配结果的方式更灵活，这正是其巧妙的地方。一个隐蔽的好处是re.sub()是每匹配一次就传递中间结果给update函数，所以update函数每次都只对一个匹配到的关键字或变量进行处理。如果是批量的替换，就会出现这样的错误：当有变量名原本就是字母时，如a,之前已经有变量被替换为a，那么已经被替换为的变量会因为新变量a而再次被替换为其它字母。输入测试中的["function where(a) return a"]就是为这种情况设立的。
